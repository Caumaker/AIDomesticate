```markdown
# API Reference

## Planner

### Methods

#### `createPlan(task)`

- **Description**: Creates a plan based on the given task.
- **Parameters**:
  - `task` (string): The task to be planned.
- **Returns**:
  - `string`: A detailed plan.

## Maker

### Methods

#### `executePlan(plan)`

- **Description**: Executes the given plan.
- **Parameters**:
  - `plan` (string): The plan to be executed.
- **Returns**:
  - `string`: The result of the execution.

## Supervisor

### Methods

#### `reviewOutput(output)`

- **Description**: Reviews the output and provides feedback.
- **Parameters**:
  - `output` (string): The output to be reviewed.
- **Returns**:
  - `string`: The feedback for improvement.
```
